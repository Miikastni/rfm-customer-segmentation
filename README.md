RFM-анализ клиентов

Задача:
Определить, какие клиенты приносят основную выручку и как они себя ведут.

Что сделал:
- Рассчитал давность, частоту и сумму покупок
- Разделил клиентов на группы
- Проанализировал вклад сегментов в выручку

Ключевые выводы:
- Небольшая доля клиентов приносит большую часть выручки
- Есть сегмент лояльных клиентов с регулярными покупками
- Много клиентов не возвращаются после первой покупки
- Есть “уснувшие” клиенты, которых можно вернуть

Рекомендации:
- Работать с удержанием ключевых клиентов
- Реактивировать неактивных через маркетинг
- Улучшать первый опыт новых пользователей
