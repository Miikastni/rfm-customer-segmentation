 1. RFM-анализ клиентов (Online Retail)

  Описание проекта

Цель проекта — сегментировать клиентов на основе их покупательского поведения с использованием метода RFM (Recency, Frequency, Monetary).



 2. Используемые инструменты

 SQL (PostgreSQL)
 DBeaver
 Excel (для визуализации)



 3. Данные

Использован датасет онлайн-ритейла, содержащий:

 заказы (InvoiceNo)
 товары
 даты покупок (InvoiceDate)
 клиентов (CustomerID)
 цены и количество



 4. Этапы анализа

 Очистка данных (удаление NULL, отрицательных значений)
 Расчет метрик:

    Recency — давность последней покупки
    Frequency — количество заказов
    Monetary — общая выручка
 Сегментация клиентов с помощью RFM
 Анализ полученных сегментов



 5. Основные результаты

 Обнаружена поляризация клиентской базы: значительная доля клиентов относится к сегментам 555 и 111
 Основная выручка формируется сегментом 555 (лояльные клиенты)
 Выявлены клиенты с высоким доходом, но низкой активностью (например, 115)
 Обнаружено большое количество неактивных клиентов



 6. Рекомендации

 Сегмент 555: удержание и увеличение среднего чека
 Сегменты 115/155: реактивация через персональные предложения
 Сегмент 111: оптимизация маркетинговых затрат или массовые кампании
